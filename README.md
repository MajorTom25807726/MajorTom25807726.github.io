波波的程式(python,需要執行請用Google Colab)
=========
```python
import random
max = 100
min = 0
number = random.randint(0, 100)
 
 
while 1:
    guess = int(input("請輸入數字:"))
    if guess > max:
        print("錯誤:輸入值超過最大值,請重新輸入!")
        guess = int(input("請輸入數字:"))
    else:
        if guess < number:
            min = guess
            print("範圍:",min,"到",max)
        elif guess > number:
            max = guess
            print("範圍:",min,"到",max)
        elif guess == number:
            break
print("正確數字為:",number)
```
